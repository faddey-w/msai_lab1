Ввод и вывод данных производится в консоли
Примеры программ, написанные на С++, приведены в данной папке
Подключение модулей производится в файле config.ini

Входные данные
<количество вершин> <количество ребер q> <количество дополнительные данных (начальная вершина, конечная вершина, т. п.)>
<описание вершины 1>
..
<описание вершины n>
<описание ребра 1>
..
<описание ребра m>
<строка 1>
..
<строка q>

Примечание: дополнительных данных может быть дано больше, чем требутся в модуле, но не меньше

<описание вершины> = 
<индекс вершины> <количество весов q>
<строка 1>
..
<строка q>

<описание ребра> = 
<1, если ребро ориентировано, 0 если не ориентировано> <начальная вершина> <конечная вершина> <количество весов q>
<строка 1>
..
<строка q>

Выходные данные
Если нужно выдать сообщение пользователю, вывести нужный текст
Если нужно продемонстрировать, вывести последовательность вершин

Примечания:
1. Если ничего не будет выведено, это будет расцениваться как последовательность из 0 вершин
2. Выводимый тест формата html, то есть можно применять язык html для изменения формата вывода (<b></b><i></i>)
